Nginx平滑升级

发表于 2017-07-28

参考：《Nginx高性能Web服务器详解》

思路如下：

  1. 备份原有的nginx目录
  2. 备份旧版本nginx主程序
  3. 编译安装新版本的nginx，使用原有的配置选项（原先部署的时候的），部署到同一个目录下
  4. 平滑升级，此时两个版本的nginx都同时运行
  5. 杀掉老的nginx进程
  6. 升级结束

具体操作如下：

​ 1.查看原有的编译选项

​ 2.备份旧版本的nginx

$sudo mv /usr/local/nginx/sbin/nginx /usr/local/nginx/sbin/nginx.oldbin

​ 3.编译新版的nginx

​ 4.平滑升级

此时，在logs目录下，之前的pid文件变为nginx.pid.oldbin：

$ ls -l ../logs/

查看nginx进程，此时有两个nginx进程同时在运行；

$ps -ef|grep nginx

关闭旧的nginx进程

参考Nginx可接收信号

TERM或 INT 快速停止nginx

QUIT 平缓停止nginx

HUP 平滑重启

USR1 重新打开日志文件，常用于日志切割

USR2 平滑升级

WINCH 平缓停止worker process，用于nginx平滑升级

Mysql5.7 重置root密码

发表于 2017-07-28

如果MySQL再运行，先杀掉mysqld进程

killall -TERM mysqld

运行mysqld_safe --skip-grant-tables &

此时如果不想被远程连接，那么执行：

mysqld_safe --skip-grant-tables --skip-networking &

使用mysql连接server

update mysql.user set authentication_string=password('123qwe') where user='root' and Host = 'localhost';

特别提醒注意的一点是，新版的mysql数据库下的user表中已经没有Password字段了,而是将加密后的用户密码存储于authentication_string字段

修改完毕。重启

然后mysql就可以连接了

提高效率的xx个App推荐

1.科学上网-Surge/ShadowRocket 推荐度：❤❤❤❤❤ 价格：18+ShadowSocks

Surge比较贵，功能太多
如果仅仅是翻墙，还是建议使用ShadowRocket比较划算，毕竟才￥18
可以尝试去App Store翻一下Surge的价格 - -！
可以参考一下App Store Surge的价格：

然而ShadowRocket的价格就亲民很多，可以使用ShadowSocks的规则，也是很稳定的，所以，如果日常使用的话，可以选择这个来进行“科学上网”

ShadowSocks服务：ShadowSocks是一个VPN服务，通过ShadowSocks可以访问国外一些不能访问的网站，如Google、Facebook、Twitter等网站；购买ShadowSocks服务，可以在各个平台通用，都可以翻墙，比如Windows、IOS、安卓、linux、浏览器插件、路由器等；购买之后，得到的是一个服务器账号和密码，需要通过其他工具配合使用，一次配置，持久使用
详细介绍请参考：ShadowSocks注册详解

2.相册备份-Google相册 推荐度：❤❤❤❤❤ 价格：免费

为什么选择Google相册呢？原因就是不用考虑空间容量的大小，高清质量图片备份无限容量
当然，这个前提是你科学上网了

3.密码管理-1Password 推荐度：❤❤❤❤❤ 价格：免费

因为现在的App，很多，动不动就需要注册，很多时候，我们都是用同一个密码登录所有的网站，但是！！！
这样实在是不安全！
因为黑客会攻击最容易攻破的网站，然后使用这个密码来撞库（就是用这个账户和密码来尝试登录其他的网站）
所以，很有必要设置一些不同的密码。
而且，很多网站，我们并不需要登录很多次，可能只需要登录几次，所以，这些密码，我们可以设置得很复杂，然后，自己都不知道，但是可以保存在1Password中，然后需要的时候，再来这里找到这个密码和账户。

当然，如果仅仅是保存密码和账户，那么简单，可能觉得一个记事本就可以了
笔者用的是Mac，有Chrome浏览器插件，当你注册一个新的网站的时候，可以选择使用1Password进行解锁，然后直接保存，无需手动保存，并且会记录当时输入密码的网站，并且，支持生成密码，按照规则生成密码

当你登录这个网站的时候，会在插件中显示这个账户，免去你去搜索这个账户

最重要的是，每次你登录1Password的时候，都需要输入密码，或者用指纹进行解锁，所以，你只需要记得一个复杂的，无法被轻易破解的密码即可。
这个工具，笔者每天都会打开，完全依赖了！

4.Wuderlist奇妙清单-清单Clear 推荐度：❤❤❤❤❤ 价格：免费

笔者用过很多的清单，无论是各种花哨的，功能强大的，但是用的最多的还是奇妙清单，并且已经把其他的都删掉了，工具，好用就OK
奇妙清单最多支持二级菜单，但是对我来说已经足够
另外支持分享清单，支持@其他人
更多的是，界面简单，多平台支持！！！
工作和个人的都可以放里面
我现在甚至是把它当做了，工作日志记录工具
并且支持话题，使用#话题#就可以设置话题，根据话题找到相关的任务，十分好用的功能
